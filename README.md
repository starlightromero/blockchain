# DeadCoin

Reviving the future of currency

## How to Run

To run the app you will need:

- [Docker](https://docs.docker.com/get-docker/)
- [docker-compse](https://docs.docker.com/compose/install/)

Then run:

```bash
make start
```

To stop the app press `CNTRL + C`. Then run:

```bash
make stop
```
